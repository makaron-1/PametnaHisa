# Pametna Hiša

## Opis projekta
Aplikacija Pametna Hiša je namizni program, razvit v okolju Windows Forms, ki omogoča upravljanje različnih pametnih naprav v domu. Sistem podpira naprave, kot so luči, alarmi, termostati, rolete in klimatske naprave.

Uporabnik lahko naprave dodaja, upravlja njihovo delovanje, spreminja nastavitve ter spremlja porabo energije.

---

## Namestitev

Za uporabo aplikacije potrebujete:

- Microsoft Visual Studio
- .NET podporo za Windows Forms

### Koraki:
1. Prenesite projekt iz GitHub repozitorija (Download ZIP)
2. Razširite ZIP datoteko
3. Odprite `.sln` datoteko v Visual Studio
4. Počakajte, da se projekt naloži

---

## Zagon aplikacije

1. V Visual Studiu kliknite **Start (F5)**
2. Aplikacija se zažene in prikaže glavni obrazec

---

## Navodila za uporabo

### Dodajanje naprave
1. Vnesite ime naprave
2. Izberite tip naprave (Luč, Alarm, Termostat, Roleta, Klima)
3. Kliknite gumb **Dodaj**

---

### Upravljanje naprav
- Kliknite **Vklopi** za vklop naprave
- Kliknite **Izbriši** za odstranitev naprave

---

### Nastavitve
1. Izberite napravo
2. Kliknite **Settings**
3. Vnesite geslo (če je potrebno)
4. Spremenite nastavitve

---

### Temperatura
- Pri termostatu in klimi se prikaže trenutna temperatura
- Barva se spremeni glede na temperaturo

---

### Poraba energije
- Prikaže se poraba posamezne naprave
- Prikaže se skupna poraba vseh naprav

---

### Primerjava naprav
1. Vnesite imeni dveh naprav
2. Izberite operator
3. Kliknite gumb za primerjavo

---

### Nočni način
- Kliknite **Night mode**
- Vse naprave se izklopijo

---

### Upravljanje vseh naprav
- Vklop vseh naprav
- Izklop vseh naprav
